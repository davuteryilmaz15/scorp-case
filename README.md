# scorp-case

scorp social media application.

## Installation

1. download [python](https://www.python.org/downloads/) and install.

## Run on your pc

clone the project

```bash
  git clone https://github.com/davuteryilmaz15/scorp-case.git
```

go to project directory

```bash
  cd scorp-case
```

initialize db

```bash
  python db.database.py
```

run application

```bash
  python app.py
```
